SQL API

SQL API reference

  • bioseq.cigarsql_functionCIGAR string describing the current mmsearch alignment.
  • bioseq.lensql_functionLength of a bioseq.seq value in biological symbols (residues).
  • bioseq.query_endsql_functionEnd offset of the current mmsearch alignment on the query sequence.
  • bioseq.query_startsql_functionStart offset of the current mmsearch alignment on the query sequence.
  • bioseq.symbol_countsql_functionNumber of occurrences of a given symbol within a bioseq.seq value.
  • bioseq.target_endsql_functionEnd offset of the current mmsearch alignment on the stored target sequence.
  • bioseq.target_startsql_functionStart offset of the current mmsearch alignment on the stored target sequence.
  • public.mmsearchsql_functionSimilarity-search predicate: true when the query sequence aligns to the target under default aligner options. The columnar custom scan rewrites this qualification at plan time.
  • public.mmsearchsql_functionSimilarity-search predicate with explicit per-call aligner options.
  • bioseq.seqsql_typeCompact, alphabet-aware biological sequence column type: a variable-length, TOAST-able value storing a nucleotide or protein sequence for indexing, similarity search, and analysis inside PostgreSQL.
  • public.bioseq_aligner_optionssql_typePer-call aligner tuning options (k-mer and window sizes, gap and chaining limits) for the mmsearch similarity predicate.
  • public.mmsearchoverload_groupOverloaded SQL function with 2 signatures.
