SQL API

  • bioseq.len sql_function — Length of a bioseq.seq value in biological symbols (residues).
  • bioseq.symbol_count sql_function — Number of occurrences of a given symbol within a bioseq.seq value.
  • public.mmsearch sql_function — Similarity-search predicate: true when the query sequence aligns to the target under default aligner options. The columnar custom scan rewrites this qualification at plan time.
  • public.mmsearch sql_function — Similarity-search predicate with explicit per-call aligner options.
  • bioseq.seq sql_type — Compact, alphabet-aware biological sequence column type: a variable-length, TOAST-able value storing a nucleotide or protein sequence for indexing, similarity search, and analysis inside PostgreSQL.
  • public.bioseq_aligner_options sql_type — Per-call aligner tuning options (k-mer and window sizes, gap and chaining limits) for the mmsearch similarity predicate.
  • public.mmsearch overload_group — Overloaded SQL function with 2 signatures.
